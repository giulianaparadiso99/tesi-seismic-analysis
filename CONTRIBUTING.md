# Contributing to Seismic Analysis Pipeline

This repository contains the analysis pipeline developed for the Master's thesis "Statistical modeling and scaling analysis of seismic acceleration signals" at Politecnico di Torino.

## Project Status

This project represents completed research for a Master's thesis in Mathematical Engineering. The core analysis pipeline is **complete and stable** as of May 2026.

## How to Use This Code

### For Reproduction

To reproduce the analysis:

1. **Clone the repository**
   ```bash
   git clone https://github.com/giulianaparadiso99/tesi-seismic-analysis.git
   cd tesi-seismic-analysis
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Download required data**
   - ITACA strong-motion database: [link in README]
   - INSTANCE metadata: http://doi.org/10.13127/instance/eventsmetadata.3

4. **Run notebooks in sequence**
   - `01a_metadata.ipynb` → `01b_signal_loading.ipynb` → `02_signal_conversion.ipynb` → etc.

### For Extension or Adaptation

If you wish to extend or adapt this code for your own research:

1. **Fork the repository** to your own GitHub account
2. **Cite appropriately** using the `CITATION.cff` file
3. **Acknowledge modifications** clearly in your documentation
4. **Respect the license** (see LICENSE file)

### Notebooks

- **Clear section headers** with numbering
