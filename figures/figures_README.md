# Analysis Figures

All figures generated during the seismic signal analysis pipeline, organized by analysis stage and signal type.

---

## Directory Structure

```
figures/
├── 01a_metadata/                        # Metadata analysis
├── 01b_signals/                         # Signal preprocessing
├── 02_pdf_analysis/                     # Probability distributions
├── 03a_phase_identification/            # AR-AIC phase detection (legacy)
├── 03a_phase_identification_ar_pick/    # AR-AIC phase detection (current)
├── 03b_phase_identification_phasenet/   # PhaseNet phase detection
├── 04a_moment_scaling_spatial/          # Spatial ensemble scaling
└── 05_sensitivity_analysis/             # Monte Carlo uncertainty
```

**Organization pattern:**
Most directories follow: `{stage}/{signal_type}/{analysis_aspect}/`

**Signal types:** `acceleration/`, `velocity/`, `displacement/`

---

## 01a. Metadata Analysis (`01a_metadata/`)

Exploratory analysis of event catalog and station metadata.

**Structure:** `01a_metadata/{signal_type}/`

**Typical figures:**
- Station maps with PGA overlay
- Epicentral distance distributions
- Magnitude-distance relationships
- Correlation matrices
- Metadata quality checks

**Generated by:** `01a_metadata.ipynb`

---

## 01b. Signal Preprocessing (`01b_signals/`)

Signal quality control and preprocessing validation.

**Structure:** `01b_signals/{signal_type}/waveforms/`

**Typical figures:**
- Raw vs. preprocessed waveforms
- Signal length distributions
- Amplitude distributions by component
- Baseline correction validation
- Normalization checks (PDF pipeline)

**Generated by:** `01b_signals.ipynb`

---

## 02. PDF Analysis (`02_pdf_analysis/`)

Probability density function characterization.

**Structure:** `02_pdf_analysis/{signal_type}/{analysis_type}/`

### Analysis Types

**`gaussian_fit/`** - Gaussian distribution fitting
- Individual signal fits (one PDF per signal)
- Q-Q plots vs. normal distribution
- Anderson-Darling test results
- Kurtosis and skewness distributions
- Summary: classification bar chart (Gaussian vs. non-Gaussian)

**`heavy_tail/`** - Heavy tail assessment
- PDF fits: Gaussian, Laplace, Student-t, Lévy stable
- Q-Q plots
- Log-log CCDF with power-law fits
- AIC/BIC model comparison
- Summary: Student-t degrees of freedom, power-law exponents

**`pdf_single/`** - Individual PDF plots
- Simple histograms per signal
- Normalized distributions

**Status:**
- `acceleration/`: Complete (3 subdirectories with figures)
- `velocity/`: Empty (not yet analyzed)
- `displacement/`: Empty (not yet analyzed)

**Generated by:** `02_pdf_analysis.ipynb`

## 03a. AR-AIC Phase Detection (Current) (`03a_phase_identification_ar_pick/`)

Autoregressive-AIC onset detection and windowing results.

**Structure:** `03a_phase_identification_ar_pick/{signal_type}/{analysis_type}/`

### Analysis Types per Signal Type

**`onset_detection/`** - P and S wave detection
- Time series with detected onsets overlaid
- Search windows visualization
- Detected vs. theoretical arrival comparison
- Residuals vs. epicentral distance
- Monotonicity checks

**`coda_detection/`** - Coda onset detection
- Comparison of four methods: Rautian, Arias, Envelope, Median
- Method agreement analysis
- Distance-dependent behavior
- Statistical validation

**`windows_rautian/`** - Four-window segmentation (Rautian coda)
- Pre-event, P-wave, S-wave, coda windows
- Grid views of multiple signals
- Individual signal plots with all windows

**`windows_arias/`** - Same structure, Arias coda method

**`windows_envelope/`** - Same structure, Envelope coda method

**`windows_median/`** - Same structure, Median coda method

**Complete for:** All three signal types (acceleration, velocity, displacement)

**Generated by:** `03_phase_identification.ipynb`

## 03b. PhaseNet Phase Detection (`03b_phase_identification_phasenet/`)

**Structure:** `03b_phase_identification_phasenet/{signal_type}/{analysis_type}/`

**`windows_{coda_method}/`** - Four-window segmentation

**Note:** No `coda_detection/` subdirectory (coda methods same as AR-AIC)

**Complete for:** All three signal types

**Generated by:** `03_phase_identification.ipynb`

## 04a. Spatial Ensemble Scaling (`04a_moment_scaling_spatial/`)

Moment scaling analysis with spatial averaging across stations.

**Structure:** `04a_moment_scaling_spatial/{picker}/{signal_type}/`

### Pickers
- `ar_pick/` - AR-AIC phase detection results
- `phasenet/` - PhaseNet phase detection results

### Typical Figures per Combination

**Scaling curves:**
- M_q(τ) vs τ in log-log space
- One curve per moment order q
- Four panels: pre-event, P-wave, S-wave, coda

**Scaling exponents:**
- ζ(q) vs q
- Comparison across windows
- Linear reference (ζ(q) = q/2 for normal diffusion)
- Error bars from fit uncertainty

**Window comparison:**
- Overlay of ζ(q) for all four windows
- Identification of anomalous diffusion regimes
- Breakpoint detection

**Q-specific analysis:**
- ζ(q=1) vs ζ(q=2) scatter plots
- Window-to-window evolution

**Status:**
- All 6 combinations complete (2 pickers × 3 signal types)
- Each contains ~5-10 summary PDFs

**Generated by:** `04a_moment_scaling_spatial.ipynb`

## 05. Sensitivity Analysis (`05_sensitivity_analysis/`)

Monte Carlo uncertainty quantification for phase pick perturbations.

**Structure:** `05_sensitivity_analysis/phasenet/{signal_type}/`

**Note:** Only PhaseNet picker analyzed (AR-AIC not yet done).

### Typical Figures

**Perturbation impact:**
- Window duration changes under noise/bias
- Peak timing shifts
- Scaling exponent variations

**Monte Carlo distributions:**
- Histogram of ζ(q) across 100 iterations
- Confidence intervals
- Sensitivity metrics (coefficient of variation)

**Comparison plots:**
- Nominal vs. perturbed ζ(q) curves
- Error propagation analysis

**Status:**
- `acceleration/`: Complete
- `displacement/`: Complete (or in progress)
- `velocity/`: Not started

**Generated by:** `05_sensitivity_analysis.ipynb`

## Usage Tips

### Finding Specific Figures

**By analysis stage:**
```bash
ls figures/03a_phase_identification_ar_pick/acceleration/onset_detection/
```

**By signal type:**
```bash
find figures/ -path "*/acceleration/*" -name "*.pdf"
```

**By coda method:**
```bash
find figures/ -path "*/windows_rautian/*" -name "*.pdf"
```

**Summary plots only:**
```bash
find figures/ -name "*summary*.pdf"
```

### Opening Figures

**Single file:**
```bash
open figures/04a_moment_scaling_spatial/ar_pick/acceleration/scaling_exponents_all_windows.pdf
```

**All in directory:**
```bash
open figures/02_pdf_analysis/acceleration/gaussian_fit/*.pdf
```

### Batch Operations

**Count PDFs per signal type:**
```bash
find figures/ -path "*/acceleration/*" -name "*.pdf" | wc -l
find figures/ -path "*/velocity/*" -name "*.pdf" | wc -l
find figures/ -path "*/displacement/*" -name "*.pdf" | wc -l
```

**Check total size:**
```bash
du -sh figures/
du -sh figures/04a_moment_scaling_spatial/
```

## Quality Standards

All figures follow these conventions:

**Axes:**
- Clear labels with units
- Appropriate scale (linear/log)
- Tick marks and grid (when helpful)

**Legends:**
- Positioned to avoid data occlusion
- Descriptive labels
- Consistent ordering across related plots

**Colors:**
- Colorblind-friendly palette
- Consistent across analysis stages
- HNE (blue), HNN (orange), HNZ (green)

**Text:**
- Readable font sizes (≥8pt)
- LaTeX-compatible fonts
- Titles indicate signal/station when relevant

**Layout:**
- Tight bounding boxes (no excess white space)
- Multi-panel figures use consistent aspect ratios
- Subplots aligned and properly spaced
