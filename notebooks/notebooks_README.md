# Analysis Notebooks

This directory contains Jupyter notebooks for seismic moment scaling analysis.

---

## 📚 Notebook Execution Order

Run notebooks in this sequence:

### 1. Data Exploration & Metadata
**`01_metadata.ipynb`** - Event catalog analysis
- Catalog statistics (magnitude distribution, temporal distribution)
- Station coverage and spatial distribution
- Data quality assessment

**Output:** Catalog summaries, metadata visualizations

---

### 2. Signal Preprocessing
**`02_signals.ipynb`** - Signal processing and quality control
- Load raw seismic data
- Baseline correction
- Event window detection (threshold-based onset detection)
- Signal quality control and validation

**Output:** 
- `data/processed/signals_long.parquet` - Full signals (baseline corrected)
- `data/processed/signals_event.parquet` - Event windows only

---

### 3. Single Signal Analysis

#### 3a. Probability Density Functions
**`03a_pdf_analysis.ipynb`** - PDF fitting and distribution analysis
- Increment distribution analysis
- Gaussian fit assessment
- Heavy tail identification
- Power-law tail fitting

**Output:** PDF plots, tail exponents

---

#### 3b. Moment Scaling (MAIN ANALYSIS)
**`03b_moment_scaling.ipynb`** - Multifractal moment scaling analysis
- Process integration (acceleration → velocity → displacement)
- Increment computation for multiple time lags τ
- q-th order moment calculation
- Scaling exponent ζ(q) estimation
- Linearity tests and piecewise fitting

**Output:**
- Scaling exponents for all processes
- Linearity and piecewise fit results
- Comprehensive scaling plots
- `data/processed/scaling_*.parquet` files

**This is the core analysis notebook for the thesis.**

---

#### 3c. Autocorrelation
**`03c_autocorrelation.ipynb`** - Temporal correlation analysis
- Autocorrelation function computation
- Correlation length estimation
- Memory effects assessment

**Output:** Autocorrelation plots and statistics

---

### 4. Aggregated Analysis

#### 4a. Ensemble Scaling
**`04a_ensemble_scaling.ipynb`** - Ensemble moment scaling
- Spatial averaging across multiple stations
- Ensemble vs time-averaged moments comparison
- Ergodicity tests

**Output:** Ensemble scaling exponents, ergodicity plots

---

#### 4b. Aggregated PDFs
**`04b_aggregated_pdf.ipynb`** - Aggregated probability distributions
- Combined PDF across all signals
- Universal distribution assessment
- Comparison with single-signal PDFs

**Output:** Aggregated PDF plots and fits

---

## 🚀 Quick Start

### First Time Setup

```bash
# Install dependencies
pip install -r requirements.txt

# Start Jupyter
jupyter notebook
```

### Run Main Analysis

For the core moment scaling analysis:

```bash
jupyter notebook 03b_moment_scaling.ipynb
```

Then execute all cells (Cell → Run All).

## 🛠️ Troubleshooting

### "Module not found" errors

```bash
# Make sure you're in the project root
cd /path/to/tesi-seismic-analysis

# Install requirements
pip install -r requirements.txt
```

### "File not found" errors

- Check that `02_signals.ipynb` has been run first
- Verify files exist in `data/processed/`

### Kernel crashes / Memory errors

- Reduce the number of signals being processed
- Close other applications
- Increase swap space if using large datasets

### Plots not showing

```python
# Add at the beginning of notebook
%matplotlib inline
```
