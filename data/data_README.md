### Signal Files

#### `signals_long.parquet`
**Description:** Full acceleration signals with baseline correction

**Columns:**
- `file` (str) - Unique file identifier (network.station.location.channel)
- `sample` (int) - Sample index
- `time` (float) - Time in seconds
- `acceleration` (float) - Baseline-corrected acceleration (cm/sВІ)
- `station` (str) - Station code
- `stream` (str) - Channel code (e.g., HNE, HNN, HNZ)

**Generated by:** `notebooks/02_signals.ipynb`

**Size:** ~[estimate size]

**Usage:**
```python

df = pd.read_parquet('data/processed/signals_long.parquet')
```

---

#### `signals_event.parquet`
**Description:** Event windows only (trimmed signals around main seismic event)

**Columns:** Same as `signals_long.parquet`

**Event Detection:** Threshold-based onset detection with STA/LTA ratio

**Generated by:** `notebooks/02_signals.ipynb`

**Usage:** Used for focused analysis on main event window

---

#### `event_window_with_processes.parquet`
**Description:** Event windows with integrated velocity and displacement

**Columns:**
- All columns from `signals_event.parquet`
- `velocity` (float) - Integrated velocity (cm/s)
- `displacement` (float) - Integrated displacement (cm)

**Integration method:** Trapezoidal rule

**Generated by:** `notebooks/03b_moment_scaling.ipynb`

**Usage:** Main analysis file for moment scaling of all three processes

## рџ’ѕ File Formats

### Parquet Format

All processed files use Apache Parquet format:

**Advantages:**
- Columnar storage (fast queries)
- Built-in compression (smaller files)
- Preserves data types
- Fast I/O with pandas

**Reading Parquet files:**
```python
import pandas as pd

# Read entire file
df = pd.read_parquet('data/processed/signals_long.parquet')

# Read specific columns (faster)
df = pd.read_parquet('data/processed/signals_long.parquet', 
                     columns=['file', 'acceleration'])

# Filter while reading (very fast)
df = pd.read_parquet('data/processed/signals_long.parquet',
                     filters=[('station', '==', 'ACER')])
```

**Writing Parquet files:**
```python
df.to_parquet('output.parquet', index=False)
```

## рџ“– Column Conventions

### Physical Units
- Acceleration: cm/sВІ
- Velocity: cm/s
- Displacement: cm
- Time: seconds
- Time lags (П„): samples (multiply by 0.005 for seconds at 200 Hz)

### Naming Conventions
- `file`: Format `network.station.location.channel`
- `station`: 4-character station code (e.g., ACER, CAMP)
- `stream`: 3-character channel code (e.g., HNE, HNN, HNZ)
