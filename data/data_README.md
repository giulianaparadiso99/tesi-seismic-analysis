## Raw Data

**Location:** `data/raw/seismic_data.zip`

**Format:** ASCII text files (`.ASC`) - One file per station/channel

- Acceleration time series from seismometers
- Sampling rate: 200 Hz
- Three-component recordings (HNE, HNN, HNZ)
**To extract:**
```bash

unzip seismic_data.zip
```

### Moment Scaling Files

All scaling files follow naming pattern: `scaling_[type]_[process]_[window].parquet`

Where:
- **type**: increments, moments, exponents, linearity, piecewise
- **process**: acc, vel, disp (acceleration, velocity, displacement)
- **window**: (nothing for full signal), `_event` (event window only)

#### Increments (6 files)

Files: `scaling_increments_[process](_event).parquet`

**Description:** Computed increments Œîx(œÑ, t‚ÇÄ) = x(t‚ÇÄ + œÑ) - x(t‚ÇÄ)

**Columns:**
- `file` (str) - File identifier
- `station` (str) - Station code
- `stream` (str) - Channel code
- `tau` (int) - Time lag (samples)
- `t0` (int) - Starting time index
- `increment` (float) - Increment value

**Files:**
- `scaling_increments_acc.parquet` - Acceleration, full signal
- `scaling_increments_acc_event.parquet` - Acceleration, event window
- `scaling_increments_vel.parquet` - Velocity, full signal
- `scaling_increments_vel_event.parquet` - Velocity, event window
- `scaling_increments_disp.parquet` - Displacement, full signal
- `scaling_increments_disp_event.parquet` - Displacement, event window

**Generated by:** `notebooks/03b_moment_scaling.ipynb`

## File Naming Conventions

### Scaling Files

- `increments` - Raw increments Œîx(œÑ, t‚ÇÄ)

**Window:**

## File Formats

### Parquet
All `.parquet` files use Apache Parquet format:

**Reading:**
```python
import pandas as pd

# Read entire file
df = pd.read_parquet('data/processed/scaling_moments_disp.parquet')

# Read specific columns
df = pd.read_parquet('data/processed/scaling_moments_disp.parquet',
                     columns=['file', 'q', 'tau', 'moment'])

# Filter while reading
df = pd.read_parquet('data/processed/scaling_moments_disp.parquet',
                     filters=[('q', '==', 2.0)])
```

### LaTeX Tables
`.tex` files are ready-to-include LaTeX tables.

**Including in thesis:**
```latex
\begin{table}
  \centering
  \input{data/processed/tables/processes_checks_table.tex}
  \caption{Process validation checks}
  \label{tab:checks}
\end{table}
```
